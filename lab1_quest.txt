Как выбрать только уникальные значения какого-либо столбца?
с помощью DISTINCT. Пример, выбрать уникальные города:
SELECT DISTINCT city FROM airports

Как осуществить сортировку по возрастанию/убыванию по значению какого-либо столбца?
с помощью ORDER BY, DESC - для сортировки в обратном порядке.
(в прямом порядке ASC, но его можно опустить, т.к. это дефолтное состояние)
Пример: обратная сортировка городов
SELECT city FROM airports ORDER BY city DESC

Как агрегатные функции ведут себя по отношению к неопределённым значениям?
Все, кроме count выдают NULL, count игнорит

Чем отличаются UNION и UNION ALL?
при соединении двух запросов дублирующиеся строки отфильтровываются, если не указано ALL

Чем отличаются COUNT(*) и COUNT(field)?
Например, count(*) подсчитает общее количество строк, а count(f1) только количество строк,
в которых f1 не NULL (так как count игнорирует NULL),
а count(distinct f1) подсчитает число различных и отличных от NULL значений столбца f1

Как подсчитать количество уникальных значений столбца?
пример, посчитаь кол-во уникальных городов, в которых есть аэропорты
SELECT count(DISTINCT city) FROM airports

Как можно осуществить проверку на неопределенное значение?
x IS UNKNOWN (для буквенных)
IS NOT NULL

Как можно выбрать только определенное количество строк?
Пример, с 5 по 15 строки таблицы airports
SELECT * FROM airports LIMIT 10 OFFSET 5
